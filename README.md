# Bike Geometry Comparison Tool

A visual tool for comparing bike geometries side by side. Includes preset libraries for popular road and gravel bikes, including Tour de France team bikes.

## Docker

Pull and run the image from Docker Hub:

```bash
docker pull reindelttt/bike-geometry-comparison:latest
docker run -p 8080:80 reindelttt/bike-geometry-comparison
```

Then open [http://localhost:8080](http://localhost:8080).

Docker Hub: [https://hub.docker.com/r/reindelttt/bike-geometry-comparison](https://hub.docker.com/r/reindelttt/bike-geometry-comparison)

## Docker Compose

```bash
docker compose up -d
```

## Local

Simply open `index.html` in a browser.
